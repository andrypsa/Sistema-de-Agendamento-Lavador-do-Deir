# PROJETO DE EXTENSÃO I - ANÁLISE E DESENVOLVIMENTO DE SISTEMAS

## Sistema Web de Agendamento para o Lavador do Deir

### Sobre o projeto

Este projeto foi desenvolvido como parte da disciplina de Projeto de Extensão do curso de Análise e Desenvolvimento de Sistemas.

O sistema foi idealizado para atender às necessidades do estabelecimento **Lavador do Deir**, localizado no estado de Goiás. O proprietário do lavajato também exerce outra atividade profissional e, em determinados momentos, precisa interromper o atendimento no lavador para prestar outros serviços. Como consequência, alguns clientes se deslocam até o estabelecimento e o encontram fechado, ocasionando perda de tempo, gastos desnecessários com combustível e aumento da emissão de poluentes.

Diante dessa necessidade, foi proposta a criação de um sistema web para gerenciamento de agendamentos, permitindo que os clientes solicitem horários previamente e que o proprietário organize melhor sua rotina de atendimentos.

---

## Objetivos

* Organizar os agendamentos de lavadas;
* Evitar deslocamentos desnecessários dos clientes;
* Otimizar a rotina do proprietário;
* Melhorar a experiência do atendimento;
* Contribuir para práticas mais sustentáveis.

## Principais funcionalidades

### Cliente

* Cadastro;
* Login;
* Agendamento de lavagens;
* Consulta do histórico;
* Visualização do programa de fidelidade;
* Consulta de cupons de desconto.

### Administrador

* Visualização dos clientes cadastrados;
* Gerenciamento dos agendamentos;
* Aprovação ou recusa de solicitações;
* Justificativa obrigatória em caso de recusa;
* Validação de cupons de desconto.

---

## Programa de fidelidade

Após 10 lavagens realizadas, o sistema gera automaticamente um cupom de **25% de desconto**, vinculado ao cliente.

---

## Outros serviços

Além da lavagem de veículos, o estabelecimento também realiza limpeza de tapetes, carpetes e outros itens sob consulta. Para verificar disponibilidade e solicitar orçamento, o cliente poderá entrar em contato diretamente pelo WhatsApp do estabelecimento.
